## 🚀 Quick Start

```bash
# 1. Install
npm install

# 2. Get your session from the pair site
#    👉 https://mias-sessions-production.up.railway.app/

# 3. Add SESSION_ID to .env
echo "SESSION_ID=your_session_here" > .env

# 4. Launch
npm start
```

## 🎛️ What the Bot Can Do

### 🟣 Button Messages
Interactive WhatsApp **buttons** users can tap instead of typing.
- Quick-reply buttons (1–3 per message)
- Custom button text + payload
- Works in DMs and groups
- Tap → bot responds instantly

### 📋 List Menus
Scrollable **list menus** with sections and rows — perfect for command catalogs.
- Multi-section menus (Economy, Group, AI, Fun, Owner…)
- Title, description and button label per list
- Tap a row → triggers the linked command
- Auto-paginated for long menus

### 💬 Smart Text Replies
Rich **text** responses with formatting, mentions, and reply context.
- *Italic*, **bold**, ~strike~, `mono` formatting
- @mentions and quoted-message replies
- Auto-react with emojis
- Multi-language friendly

### 🛡️ Anti-Abuse Suite
Anti-link, anti-spam, anti-tag, anti-delete, anti-bot, anti-call.

### 👥 Group Tools
Promote / demote, kick, mute, welcome & goodbye, group lock, tagall, hidetag.

### 💰 Economy
Daily, weekly, work, bank, transfer, leaderboard, shop.

### 🤖 AI
Chat, image generation, image-to-text, translation, summarize.

### 🎉 Fun & Media
Stickers, memes, games, downloaders (TikTok / IG / YT / FB), audio editing.

### 🔐 Owner Controls
Eval, broadcast, ban/unban, mode (public/private), maintenance.

## ⚙️ Environment Variables

| Key | Required | Description |
|---|---|---|
| `SESSION_ID` | ✅ | From the pair site |
| `PREFIX` | ❌ | Command prefix (default `.`) |
| `OWNER_NUMBER` | ✅ | Your number with country code |
| `BOT_NAME` | ❌ | Display name |
| `MODE` | ❌ | `public` or `private` |
| `PORT` | ❌ | Web port (default `3000`) |

Steps on the site:
1. Enter your WhatsApp number (with country code, no `+`)
2. Get a **pairing code**
3. WhatsApp → **Linked Devices → Link with phone number**
4. Enter the code → copy your `SESSION_ID`
5. Paste into `.env` and run `npm start`
